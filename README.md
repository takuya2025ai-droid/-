# ビジネスコーチング・成功者マインド ナレッジダイジェスト

米国・英国のビジネスコーチング/成功者マインドセット関連ニュース・記事を毎朝自動収集し、日本語で要約して蓄積するナレッジベースです。

コーチ教育の教材ネタ・知識資産として再利用することを目的としています。

## 使い方

- 毎朝8時(日本時間)に自動で新しい要約が1件、`digests/` フォルダに追加されます
- 全体の一覧・テーマ別索引は [`digests/INDEX.md`](digests/INDEX.md) から確認できます
- 各要約ファイルには、記事タイトル・出典・元記事URL・タグ・日本語要約が記載されています
- ブラウザでこのリポジトリを開くだけで閲覧できます(特別なツールは不要です)

## フォルダ構成

```
digests/
  INDEX.md          ← 全件の索引(日付・テーマ・リンク)
  2026-07-14.md      ← 日付ごとの要約ファイル
  2026-07-15.md
  ...
```

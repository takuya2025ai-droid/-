# コーチ事業用 ナレッジダイジェスト

コーチング事業(高単価コーチング商品・Instagram発信・個別相談)を行うたくやさん向けのナレッジベースです。
毎日、3つの時間帯・ジャンルで最新の情報を自動収集・要約し、教材ネタ・Instagram投稿ネタ・セールストークのネタとして蓄積します。

## 配信スケジュール(日本時間)

| 時間 | ジャンル | 内容 | 保存先 |
|------|------|------|--------|
| 毎朝 8:00 | 📱 SNS発信テクニック | Instagram等SNS発信者向けの最新テクニック・アルゴリズム動向・バズ手法 | `digests/sns/` |
| 毎日 13:00 | 🎯 コーチングスキル | コーチング手法・心理学・認知科学など、コーチ業の方向け実践知見 | `digests/coaching/` |
| 毎日 18:00 | 🧠 マインドセット | 自己商発・成功者マインドセット。コーチ業のモチベーション・メンタルを整える情報 | `digests/mindset/` |

各回、**公開から日の浅い最新情報のみ**を2～3件収集・要約します(件数より鮮度を優先)。

## 使い方

- 全件の一覧・ジャンル別索引は [`digests/INDEX.md`](digests/INDEX.md) から確認できます
- 各要約ファイルには、タイトル・出典・元URL・タグ・日本語要約・活用ヒントが記載されています
- ブラウザでこのリポジトリを開くだけで閲覧できます
- 2026-07-13〜07-15の旧形式(全ジャンル混在)は `digests/` 直下に過去ログとして残してあります

## フォルダ構成

```
digests/
  INDEX.md
  sns/YYYY-MM-DD.md       ← 8:00 SNS発信テクニック
  coaching/YYYY-MM-DD.md  ← 13:00 コーチングスキル
  mindset/YYYY-MM-DD.md   ← 18:00 マインドセット
  2026-07-13.md ~ 2026-07-15.md ← 旧形式(参考用)
```
